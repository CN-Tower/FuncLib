funclib.js

funclib.js 是一个简单、易用、贴进业务逻辑的JavaScript UMD函数库，类似于 underscore 和 lodash。 跟它们对比，funclib.js省去了很多js自带的或者不常用的方法。

funclib.js还根据一些实际业务的逻辑的规律封装了很多前后端都实用的新方法， 让你提高代码质量和交付效率的同时，享受编程的乐趣!

funclib.js使用 TypeScript 进行开发，使用 WebPack 进行打包编译，使用 Karma / Mocha 进行单元测试， 使用 Trivas-CI 进行持续集成。

如果你喜欢funclib.js，就到GitHub上去Star或Fork吧： http://github.com/CN-Tower/funclib.js 欢迎一起来玩！

Type

fn.typeOf
检查值的类型，返回布尔值
fn.typeOf(value: any, type: 'arr' | 'obj' | 'fun' | string | string[]): boolean;
@param value [any]
接收一个任意类型的值
@param type ['arr'|'obj'|'fun'|'str'|'num'|'udf'|string|string[]]
接收一个类型字符串,可以是类型简写（如："udf" 对应 "undefined"），也可以是js标准类型的 全称（如："string"），或者类型的数组（如：['str, arr']） 

fn.typeOf('test', 'str');     // true
fn.typeOf('test', 'string');  // true
fn.typeOf(() => {}, 'fun');   // true
fn.typeOf([], 'arr');         // true
fn.typeOf({}, 'obj');         // true
fn.typeOf([], 'obj');         // false
fn.typeOf([], 'object');      // true
fn.typeOf(null, 'obj');       // false
fn.typeOf(null, 'object');    // true
fn.typeOf(undefined, ['str', 'udf']); // true

fn.array(length: number, value?: any): any[];

fn.find

@param src [any[]]

@param predicate [object|function]

fn.throttle
节流函数，适用于限制resize和scroll等函数的调用频率
fn.throttle(func: Function, wait: number, options?): Function;
@param func [function]
需要节流的函数
@param wait [number]
节流时间
@param options? [{ leading?: boolean, trailing?: boolean }] (可选)
leading: 开始时是否立即调用; trailing: 结束时是否调用 

let i = 0, duration = 500;
const log = fn.throttle(() => fn.log(i, 'throttle'), 2000);

function doLog() {
  fn.timeout(duration, () => {
    i ++;
    log();
    if (i === 8) duration = 2500;
    doLog();
  });
}

Tools

fn.rd
读文件
fn.rd(file: string): string;
@param file [string]
文件路径 

fn.rd('C: test.txt');

fn.wt
写文件
fn.wt(file: string, text: string, flag?: 'w' | 'a'): void;
@param file [string]
文件路径
@param text [string]
文件内容
@param flag ['w'|'a']
w：重写，a：追加 

fn.wt('C: test.txt', 'Test file');

fn.cp
复制文件或文件夹
fn.cp(src: string, dist: string): void;
@param src [string]
原文件或文件夹路径
@param dist [string]
目标文件或文件夹路径 

fn.cp('C: test.txt', 'D:');

fn.mv
移动文件或文件夹
fn.mv(src: string, dist: string): void;
@param src [string]
原文件或文件夹路径
@param dist [string]
目标文件或文件夹路径 

fn.mv('C: test.txt', 'D: text-file\');

fn.rm
删除文件或文件夹
fn.rm(src: string): void;
@param src [string]
目标文件或文件夹路径 

fn.rm('C: test.txt');

fn.mk
创建文件夹（注：创建文件使用fn.wt）
fn.mk(dist: string): void;
@param src [string]
文件夹路径 

fn.mk('C: test-dir');

Progress

fn.progress.start
开启进度，并传入参数
fn.start(options: any): void;
@param options [string|object]
如果是string，则为旋转头进度，如果是object则为进度条
进度条参数：{title: string, width: number = 40} 

fn.progress.start('Test');

fn.progress.stop
结束进度，结束后触发回调
fn.stop(onStopped?: Function): void;
@param onStopped [function] (可选)
结束时的回调 

fn.progress.stop(() => { // Do something... });
