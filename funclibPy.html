funclib.py

funclib.py 是一个简单、易用、贴进业务逻辑的JavaScript UMD函数库，类似于 underscore 和 lodash。 跟它们对比，funclib.py省去了很多js自带的或者不常用的方法。

funclib.py还根据一些实际业务的逻辑的规律封装了很多前后端都实用的新方法， 让你提高代码质量和交付效率的同时，享受编程的乐趣!

funclib.py使用TypeScript进行开发，使用 WebPack进行打包编译，使用 Karma / Mocha进行单元测试， 使用Trivas-CI进行持续集成。

如果你喜欢funclib.py，就到GitHub上去Star或Fork吧： http://github.com/CN-Tower/funclib.py 欢迎一起来玩！

Type

fn.typeOf
检查值的类型，返回布尔值
fn.typeOf(value: any, type: 'arr' | 'obj' | 'fun' | string | string[]): boolean;
@param value
接收一个任意类型的值
@param type ['arr'|'obj'|'fun'|'str'|'num'|'udf'|string|string[]]
接收一个类型字符串,可以是类型简写（如："udf": "undefined"），也可以是js标准类型的 全称（如："string"），或者类型的数组（如：['str, arr']） 

  fn.typeOf(true, 'bol');         // true
  fn.typeOf(true, 'boolean');     // true
  fn.typeOf([], ['arr, 'str']);   // true
  fn.typeValue([], ['obj']);      // false
